# MOP 管理后台 - 角色与权限说明

## 角色类型（共 3 类）

| 角色 | 说明 |
|------|------|
| **超级管理员** | 可使用完整后台功能 |
| **管理员** | 受限后台功能，见下表 |
| **用户** | 仅能邀请用户注册，不能登录后台 |

---

## 权限矩阵

| 项目 | 超级管理员 | 管理员 | 用户 |
|------|------------|--------|------|
| 登录后台 | ✅ | ✅ | ❌ |
| 首页 | ✅ | ✅ | - |
| 设备列表 | ✅ 全部 | ✅ 仅好友的设备与用户信息 | - |
| 用户列表 | ✅ | ❌ 不显示 | - |
| 关系列表 | ✅ | ❌ 不显示 | - |
| 应用下载 | ✅ | ✅ | - |
| 修改密码 | ✅ | ✅ 仅能改自己的密码 | - |
| 添加用户 | ✅ | ✅ | - |
| 创建管理员 | ✅ | ❌ | ❌ |
| 创建用户 | ✅ | ✅ | ❌ |
| 邀请用户注册 | ✅ | ✅ | ✅（仅 App 侧） |
| 精确查找非好友用户 | ✅ | ✅ | - |

---

## 规则说明

### 超级管理员
- 可使用完整后台功能。
- 设备列表显示全部设备与用户。
- 可创建管理员、可创建用户；可修改自己及其他账号密码（按业务需要）。

### 管理员
- **不显示的页面**：用户列表、关系列表；不显示「创建管理员」相关入口。
- **设备列表**：仅显示「好友」的设备与用户信息；可通过精确查找（如手机号/UID）查询非好友用户。
- **创建**：不能创建管理员，仅可创建用户、邀请用户注册。
- **密码**：仅能修改自己的密码。

### 用户
- 不能登录后台。
- 仅在 App 侧可邀请用户注册。

---

## 实施要点

- **好友**：管理员的好友 = 该管理员有权限查看其设备/用户信息的用户集合（如：被该管理员邀请/创建的用户，或由超级管理员分配的关系）。
- **精确查找**：管理员仅能按精确条件（手机号、UID 等）查询单条非好友用户，不提供按模糊条件列出所有非好友。
- **前端**：根据当前登录角色隐藏/展示侧栏「用户列表」「关系列表」；添加用户页中，超级管理员可选择「创建为管理员/用户」，管理员仅能创建用户（不展示角色选择）。
- **预览**：登录页提供「登录后角色（演示）」下拉框，可选超级管理员/管理员，便于验证两种角色的界面与权限差异。管理员直接访问 `users.html` 或 `relations.html` 会被重定向至首页。
